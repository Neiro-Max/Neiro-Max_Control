# Neiro-Max Control

Личный таск-менеджер с документами, Excel-таблицами, ОТЕЛЕМ (реестр гостей), договорами и медкартами — всё в одном файле.

## Как разместить на GitHub Pages (бесплатно)

1. Создайте новый репозиторий на GitHub (Settings → New repository)
2. Загрузите `index.html` и `README.md` в этот репозиторий (Add file → Upload files)
3. Перейдите в Settings репозитория → Pages
4. В разделе "Source" выберите ветку `main` и папку `/ (root)`, нажмите Save
5. Через 1-2 минуты сайт будет доступен по адресу вида:
   `https://ваш-логин.github.io/название-репозитория/`

## Важно

- Все данные сохраняются в браузере того устройства, с которого их вводили (localStorage) — они не синхронизируются между разными браузерами/телефонами/компьютерами автоматически.
- Регулярно делайте резервные копии через кнопки "Скачать резервную копию" внутри приложения — это единственный способ перенести данные на другое устройство или восстановить их при случайной очистке браузера.
